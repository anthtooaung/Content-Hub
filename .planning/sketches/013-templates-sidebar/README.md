# Sketch 013: Templates with App Shell Sidebar

## Design Question
How should the Templates page use the existing app shell sidebar for navigation?

## How to View
open .planning/sketches/013-templates-sidebar/index.html

## Variant
- **A: App Shell + Filter Bar** — Uses the existing sidebar navigation (Templates highlighted as active). Platform filtering moves to a horizontal filter bar at the top of the content area with pill-style buttons showing platform dots and counts.

## Key Design Decisions
- **Existing sidebar** — No new sidebar needed; Templates page uses the same app shell sidebar as other pages
- **Horizontal filter bar** — Platform filtering (All, TikTok, Instagram, Facebook) as pills with counts, placed above the template grid
- **Asymmetric cards** — Uses the winning Variant C design from Sketch 014 (large platform-branded visual header + clean content area)
- **Mobile bottom nav** — Three main actions: Generate, Templates (active), Dashboard
- **Responsive grid** — 3-column on desktop, adjusts for smaller screens
