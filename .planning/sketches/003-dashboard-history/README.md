# Sketch 003: Dashboard History

## Design Question
What does the saved content list feel like when populated vs empty?

## How to View
open .planning/sketches/003-dashboard-history/index.html

## Variants
- **A: Grid Cards** — Responsive card grid with platform badges and captions
- **B: List Rows** — Compact table layout, data-dense and scannable
- **C: Timeline** — Date-branching timeline with full campaign summaries

## What to Look For
- Can I quickly scan and find a past campaign?
- Does the empty state feel encouraging, not discouraging?
- Is platform information immediately visible?
- Which layout scales better with many saved items?
