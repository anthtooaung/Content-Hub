# Sketch 001: Landing Page

## Design Question
Does the hero and feature navigation feel like a calm, professional AI tool?

## How to View
open .planning/sketches/001-landing-page/index.html

## Variants
- **A: Centered Hero** -- Classic centered hero with value prop and feature grid
- **B: Split Layout** -- Left copy + right product mockup, more preview-oriented
- **C: Product-First** -- Real UI mockup front and center, how-it-works flow

## What to Look For
- First impression: does it feel professional and calming?
- Is the value proposition clear within 3 seconds?
- Does the CTA feel inviting, not aggressive?
- Are the platform icons recognizable?
