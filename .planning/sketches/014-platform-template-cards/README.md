# Sketch 014: Platform Template Cards

## Design Question
How should template cards reflect each platform's brand identity?

## How to View
open .planning/sketches/014-platform-template-cards/index.html

## Variants
- **A: Full Brand Immersion** — Each card fully embraces the platform's visual identity with gradients, branded actions (Facebook's Like/Comment/Share), and platform-specific metadata
- **B: Minimal Platform Accents** — Clean, consistent card layout with subtle platform-colored gradient borders and branded badges. More uniform across platforms.
- **C: Asymmetric Visual** — Large platform-branded visual header (120px tall gradient with giant platform initials) with clean content area below

## What to Look For
- **Brand recognition** — Can you instantly tell which platform a card is for?
- **Visual consistency** — Do the cards feel like they belong together despite different branding?
- **Readability** — Is the template title and description easy to read against the platform colors?
- **CTA clarity** — Does the "Use Template" button stand out appropriately?
- **Platform-specific details** — Facebook's social actions, TikTok's duration hints, Instagram's slide counts
