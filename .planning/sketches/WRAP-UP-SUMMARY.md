## Design Direction
Calm, modern, helpful AI content generation workspace. Blue primary (#2563EB) for AI actions, slate surfaces (#F8FAFC page, #FFFFFF cards), platform-specific colors. Sidebar navigation on desktop, bottom tabs on mobile. Split-layout auth. Tabbed post-generation results with copy as primary action. Full-page error pages with calm tone.

## Key Decisions
- **Navigation:** Sidebar (desktop) with icon+label items, section grouping. Bottom tabs (mobile) for thumb reach.
- **Auth:** Split layout with illustration panel communicating value. Social login buttons prominent.
- **Post-Gen:** Tabbed by platform. Copy is primary. Edit modal with character count. Regenerate per platform.
- **Errors:** Full-page for critical states. Calm tone. Error IDs. Retry always prominent.
- **Loading:** Progress overlay with status messages. Skeleton placeholders.
- **Forms:** Wizard flow (form → loading → results). Step indicator on mobile.
- **Palette:** Blue primary, slate surfaces, platform colors. 8px spacing scale.
