## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #F8FAFC (page bg) + #FFFFFF (cards/surfaces) | Page background, card backgrounds, form panels, inputs |
| Secondary (30%) | #E2E8F0 (borders), #F1F5F9 (subtle surface), #CBD5E1 (strong borders) | Borders, subtle backgrounds, dividers, skeleton placeholders |
| Accent (10%) | #2563EB (primary blue) | CTA buttons, active nav state, focus rings, primary links, step circles, logo icon |

**Accent reserved for:**
- Primary CTA buttons (hero + header "Sign in")
- Active nav link state (primary-soft bg + primary text)
- Focus rings (3px blue glow via `--shadow-focus`)
- How-it-works step number circles (48px, primary-soft bg, primary text)
- Logo icon square (32px, primary bg)

**Semantic colors (defined in theme, not part of 60/30/10):**
- Success: #15803D / soft #F0FDF4 (not used in Phase 1 — reserved for Phase 2/3)
- Warning: #B45309 / soft #FFFBEB (not used in Phase 1)
- Error: #B91C1C / soft #FEF2F2 (auth validation errors only)
- Platform: TikTok #111827, Instagram #E1306C, Facebook #1877F2 (not used in Phase 1 — hero mockup uses these as static display)

**Platform colors in hero mockup:** The static mockup inside the hero shows platform badges. These use platform-specific colors as decorative display, not interactive accent.

## Interaction Contracts

| Interaction | Trigger | Behavior |
|-------------|---------|----------|
| CTA hover | Mouse over primary button | Background darkens to `--color-primary-hover` (#1D4ED8), translateY(-1px) 150ms |
| CTA active | Mouse down on primary button | Background to `--color-primary-active` (#1E40AF), translateY(0) |
| Focus ring | Tab/keyboard focus on any interactive element | 3px blue glow via `--shadow-focus` (0 0 0 3px rgba(59,130,246,0.22)) |
| Nav link hover | Mouse over nav link | Color transitions to primary blue (150ms) |
| Active nav | Current page match | Primary-soft bg + primary text + control radius (10px) |
| Auth form submit | Click submit button | Button disabled, opacity 0.7, pointer-events none, loading text |
| Auth error display | API returns error | Error message appears below form with `--color-error` text, `--color-error-soft` bg, 4px top padding |
| Page transition | Navigate between pages | Standard Next.js client-side navigation (no custom transitions for Phase 1) |

## Visual Hierarchy

**Landing page focal point:** The hero heading (36px/600, "Generate social media content in seconds") is the primary visual anchor. Visual priority chain:
