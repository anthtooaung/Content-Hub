# Sketch 007: Auth Screens

## Design Question
What do login/signup feel like in the calm, professional aesthetic?

## How to View
open .planning/sketches/007-auth-screens/index.html

## Variants
- **A: Split Layout** — Left illustration panel with value prop, right side has login form with social buttons
- **B: Centered Card** — Centered card with signup form, social buttons above email fields
- **C: Minimal + Magic Link** — Passwordless login via magic link, minimal form, social fallback

## What to Look For
- Does the auth screen build trust or feel sketchy?
- Is the value proposition visible during signup (why should I create an account)?
- Are social login options prominent enough?
- Does the password strength indicator feel helpful or annoying?
- Which approach feels fastest to get started?
- Does the magic link approach feel modern or confusing?
- Use the password field to test the strength indicator (try weak/medium/strong passwords)
