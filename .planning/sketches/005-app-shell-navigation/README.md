# Sketch 005: App Shell & Navigation

## Design Question
How do the header, nav, and page transitions feel between Generate and Dashboard?

## How to View
open .planning/sketches/005-app-shell-navigation/index.html

## Variants
- **A: Top Nav (Underline)** — Classic horizontal nav with underline active state, fixed header
- **B: Sidebar** — Left sidebar with icon+label items, sticky, content area to the right
- **C: Compact + Slide-out** — Minimal header with hamburger menu, slide-out nav panel for mobile-first feel

## What to Look For
- Does the navigation feel intuitive for switching between Generate and Dashboard?
- Is the active state clear enough to know where you are?
- How does each approach handle the logo + nav + actions balance?
- Does the sidebar variant feel too heavy for a simple 2-page app?
- Does the slide-out feel natural or like unnecessary friction?
- Use the page navigation buttons to switch between Generate and Dashboard views
