# Sketch 008: Post-Generation Interactions

## Design Question
What happens after content is generated — copy, save, edit, share?

## How to View
open .planning/sketches/008-post-generation/index.html

## Variants
- **A: Card Actions** — Each platform result is a card with copy/edit/save/regenerate buttons, sticky bottom bar with Copy All
- **B: Accordion + Bulk** — Collapsible platform sections, compact view with quick actions, bulk operations at bottom
- **C: Tabbed by Platform** — Tab switcher between platforms, each tab shows full post with actions

## What to Look For
- Is copy-to-clipboard the most prominent action? (it should be)
- Does the edit flow feel natural or like an afterthought?
- Is "Regenerate" discoverable but not distracting?
- Does Save feel useful or like clutter?
- Which layout lets you scan all 3 posts fastest?
- How does the success confirmation feel?
- Use the copy buttons to test the feedback animation
- Click Edit to test the modal edit experience
- Click accordion headers in Variant B to expand/collapse
